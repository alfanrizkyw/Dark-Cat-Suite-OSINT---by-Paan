ğŸš€ Panduan Instalasi & Eksekusi Dark-Cat-Suite-OSINT
Pastikan Anda memiliki koneksi internet yang stabil dan Git serta Python 3.8+ sudah terinstal di sistem Anda.

Tahap 1: Kloning Repositori (Mengambil Kode)
Buka Command Prompt (CMD):

Kloning Repositori: Gunakan perintah git clone untuk mengunduh seluruh proyek ke komputer Anda.

Bash

git clone https://github.com/alfanrizkyw/Dark-Cat-Suite-OSINT---by-Paan.git
Masuk ke Direktori Proyek: Ganti direktori kerja Anda ke folder proyek yang baru diunduh.

Bash

cd Dark-Cat-Suite-OSINT---by-Paan
Pastikan prompt CMD Anda sekarang menunjukkan jalur yang berakhir dengan ...Dark-Cat-Suite-OSINT---by-Paan> .

Tahap 2: Instalasi Dependensi (Persyaratan)
Ini adalah langkah krusial. Kita perlu memastikan semua modul Python yang dibutuhkan oleh alat ini terinstal.

Verifikasi Python 3: Pastikan Anda menggunakan Python 3.8 atau yang lebih baru.

Bash

python --version
(Jika perintah di atas gagal, coba py -3 --version atau pastikan Python ada di PATH Anda.)

Instal Dependensi: Jalankan perintah berikut untuk menginstal semua modul dari file requirements.txt.

Bash

pip install -r requirements.txt
(Jika Anda memiliki masalah dengan versi Python, coba gunakan pip3 install -r requirements.txt.)

âš ï¸ Catatan Penting: Perhatikan jika ada pesan error berwarna merah selama proses ini (misalnya, terkait dengan "failed building wheel"). Ini berarti ada dependensi yang tidak dapat diinstal. Jika itu terjadi, Anda mungkin perlu menginstal Visual C++ Build Tools dari Microsoft.

Tahap 3: Menjalankan Suite OSINT
Eksekusi Skrip Utama: Pastikan Anda masih berada di dalam direktori proyek (Dark-Cat-Suite-OSINT---by-Paan).

Bash

python dark_cat_osint.py
(Jika python gagal, coba ganti dengan python3 atau py tergantung konfigurasi sistem Anda.)

Akses Menu: Jika berhasil, Anda akan melihat menu interaktif dengan 22+ alat yang tersedia.
